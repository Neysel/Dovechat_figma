# Система обработки ошибок DoveChat

## Обзор

В приложении реализована комплексная система обработки ошибок с красивым пользовательским интерфейсом в стиле бренда DoveChat.

## Компоненты

### 1. ErrorBoundary

React Error Boundary, который перехватывает все необработанные ошибки в компонентах и показывает красивую страницу ошибки с изображением голубя.

**Особенности:**
- Автоматически перехватывает ошибки в дочерних компонентах
- Показывает пользовательский интерфейс с брендированным дизайном
- Отображает изображение белого голубя (символ DoveChat)
- Предоставляет технические детали в раскрывающемся блоке
- Две кнопки действий: "Перезагрузить страницу" и "Попробовать снова"
- Контактная информация для поддержки

**Расположение:** `/src/app/components/ErrorBoundary.tsx`

### 2. Toaster (Toast-уведомления)

Компонент для показа всплывающих уведомлений пользователю с использованием библиотеки Sonner.

**Расположение:** `/src/app/components/Toaster.tsx`

### 3. Утилиты уведомлений

Хелперы для удобного показа toast-уведомлений.

**Расположение:** `/src/app/utils/notifications.ts`

## Использование

### Toast-уведомления

```tsx
import { notifications } from "../utils/notifications";

// Успешное уведомление
notifications.success("Операция выполнена", "Данные успешно сохранены");

// Уведомление об ошибке
notifications.error("Ошибка", "Не удалось загрузить данные");

// Информационное уведомление
notifications.info("Информация", "Новая версия доступна");

// Предупреждение
notifications.warning("Внимание", "Срок действия токена истекает");

// Уведомление о загрузке
const toastId = notifications.loading("Загрузка данных...");

// Promise-уведомление (автоматически обновляется)
notifications.promise(
  fetchData(),
  {
    loading: "Загрузка данных...",
    success: "Данные загружены!",
    error: "Ошибка при загрузке данных"
  }
);
```

### Примеры в коде

**Hero.tsx** - демонстрирует использование toast-уведомлений при клике на кнопки:

```tsx
const handleStartFree = () => {
  notifications.success(
    "Добро пожаловать в DoveChat!",
    "Начните объединять все ваши мессенджеры в одном месте"
  );
};
```

### Тестирование ErrorBoundary

Для тестирования страницы ошибки вы можете временно добавить в любой компонент:

```tsx
const TestError = () => {
  throw new Error("Тестовая ошибка для проверки ErrorBoundary");
  return null;
};

// Использование:
<TestError />
```

## Стилизация

Все компоненты обработки ошибок используют фирменные цвета DoveChat:
- Основной цвет: `#4A7FFF`
- Градиенты и переходы
- Закругленные углы в стиле современного UI

## Изображения

- **Страница ошибки:** Использует изображение белого голубя от Unsplash
- **ImageWithFallback:** Автоматический fallback для изображений, которые не загрузились

## Расширение

### Добавление логирования ошибок

Вы можете расширить `ErrorBoundary` для отправки ошибок в сервис мониторинга:

```tsx
public componentDidCatch(error: Error, errorInfo: ErrorInfo) {
  console.error('Uncaught error:', error, errorInfo);
  
  // Отправка в Sentry, LogRocket и т.д.
  // sendToErrorMonitoring(error, errorInfo);
}
```

### Кастомизация Toast-уведомлений

Отредактируйте `/src/app/components/Toaster.tsx` для изменения позиции, длительности и стилей:

```tsx
<SonnerToaster
  position="bottom-center" // изменить позицию
  toastOptions={{
    duration: 5000, // изменить длительность
    // ... другие опции
  }}
/>
```
